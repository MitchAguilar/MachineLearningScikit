# Machine Learning Scikit

Clasificador de frutas mediante aprendizaje supervisado

```sh
$ python3 -m venv myvenv
$ . myvenv/bin/activate
$ pip install -U scikit-learn
$ pip install -U numpy
$ pip install -U scipy
$ pip install -U pydot 
$ python hello.py

$ pip install pylint # Optional for Visual Code
```
